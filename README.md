# Visual Pulse — Sound-Reactive Neon Ring + Shockwave (Processing 4)

Live audio becomes glowing rings that swell with loudness and a smooth shockwave on peaks.  
Built in **Processing 4** with the **processing.sound** library.

## How to Run

1. Install **Processing 4**.
2. Install library: `Sketch → Manage Libraries… → search "Sound" → Install`.
3. Open `VisualPulse.pde` and click **Run**.

> If the mic is blocked by your OS, allow microphone access for Processing.

---

## Controls (on-screen HUD shows these live)

- **B** — toggle drum demo  
- **M** — toggle microphone  
- **↑ / ↓** — BPM  
- **Mouse click** — trigger visual pulse

## Requirements

- Processing 4.x  
- Library: **processing.sound** (AudioIn, Amplitude, Env, SinOsc, WhiteNoise)
